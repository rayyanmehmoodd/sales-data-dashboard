Abstract Deployment

This HTML is self-contained and can be hosted on GitHub Pages. To deploy, push this file and your CSV to a repository and enable Pages from the `main` branch (root).

  1. Commit `student_dashboard.html` + `student_performance_finalscore.csv` to repo.
  2. Go to repository → Settings → Pages → set source to `main` branch → Save.
  3. Your dashboard will be available at `https://.github.io//student_dashboard.html`.
Interaction types implemented: Select, Filter, Encode, Connect (brush), Reconfigure (axis swap), Elaborate (details panel), Abstract Deployment (instructions + download button).
